# Nepal Digital Platform 🇳🇵

राजनीति बुझौं, सेवा सुधारौं - डिजिटल नेपालको यात्रामा सामेल होऔं

## Features

### 🎭 Political Education Game - के होला त देशमा अब?
- Play as Nepal's political leaders (Prachanda, Deuba, Oli, etc.)
- Real-world political scenarios and decision making
- Learn about Nepal's constitution and governance

### 🏛️ Government Office Tracker - कार्यालयको अनुभव साझा गरौं  
- Track your experience in government offices
- Report waiting times, service quality, and corruption
- Help improve Nepal's public services

## Technology
- Pure HTML, CSS, JavaScript
- Mobile responsive design
- Nepali language support
- No backend required - fully static
